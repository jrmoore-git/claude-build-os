# Design — AI Slop Detection

## AI Slop Checklist

**Any listed anti-pattern detected => rewrite.**

1. Purple/violet gradient backgrounds
2. 3-column feature grid (icon-in-circle + title + description x3)
3. Icons in colored circles as decoration
4. Centered everything (`text-align: center` on all elements)
5. Uniform bubbly border-radius (same large radius on everything)
6. Generic hero copy ("Welcome to...", "Unlock the power of...", "Your all-in-one solution")
7. Body text font-size under 16px
8. More than 3 font families
9. Missing hover/focus states on interactive elements
10. `outline: none` without replacement focus indicator
